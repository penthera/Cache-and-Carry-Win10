Parameters

assetId
Type: string

(Required) A unique identifier for the asset. Used in all log events.

description
Type: string

(Required) A description of the asset. CNC only uses this in log output.

fileUrl
Type: string

(Required) The remote URL for the file (where to download from).

expectedMD5
Type: string

(Optional) CNC will validate the observed md5 of the downloaded file against this expected hash value. In case of mismatch, CNC will issue a warning and retry the download. Nil means don't validate.

permittedMIMETypes
Type: List<string>

(Required) An array of NSString mime type values. CNC will validate that the observed mime type is contained in this list, or it will issue an error and cancel the download. An empty list means "do not validate mime type."

metadata
Type: string

(Optional) A convenience field allowing you to associate arbitrary data with an asset. CNC will serialize this data and store it, but not explicitly use this data.

publishDate
Type: Nullable<DateTime>

(Optional) CNC will not provide API access to the asset until this date. Null means "now."

expiryDate
Type: Nullable<DateTime>

(Optional) CNC will not provide API access to the asset after this date. Null means no expiry.

expiryAfterPlay
Type: Nullable<long>

(Optional) Amount of time after the asset is first played that CNC will delete the asset. In seconds. Less than or equal to 0.0 means no expiry.

expiryAfterDownload
Type: Nullable<long>

(Optional) Amount of time after CNC completes download of file that CNC will delete the file. In seconds. Less than or equal to 0.0 means no expiry.

assetType
Type: EAssetType

THe type of asset.

Return Value

Type: IAsset

A new Asset object, initialized with the provided values
