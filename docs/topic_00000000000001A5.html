Remarks
A singleton object that handles all download activity and monitoring conditions and settings. Maintains the download queue, which is a list of VirtuosoAsset objects enqueued for download. Receives config values from the Backplane. Monitors device state. Ensures that downloads only occur consistent with the current configuration. <note type="warning"> You must never instantiate an object of this type directly. The VirtuosoDownloadEngine is a singleton and should only be accessed via the provided instance method. </note>
