Name Description
ImplementationProviderAttribute(string, string) This assembly attribute provides the information for instantiating an Implementation. The positional parameters specify the class and the method to be used as the factory method. The class name must be the fully qualified of name the class e.g.: Mycompany.MimeTypeValidationImplementation.MimeTypeValidation The method name is the name of the method that can instantiate the implementation. The Method MUST be a public static method with NO parameters and return an instance from which the IMimeTypeValidation is assignable from. for example: namespace MyCompany.MimeTypeValidationImplementation { public class MimeTypeValidation : IMimeTypeValidation { public static MimeTypeValidation ImplementationInstance(){return new MimeTypeValidation();} } } should have the ImplementationProviderAttribute set as [assembly: ImplementationProvider("Mycompany.MimeTypeValidationImplementation.MimeTypeValidation","ImplementationInstance")]
